# 查理·芒格 · 思维操作系统 (Munger Perspective Skill)

> "It is remarkable how much long-term advantage people like us have gotten by trying to be consistently not stupid, instead of trying to be very intelligent."

基于 73 个一手来源文件（演讲、股东会纪要、股东信、文章）深度蒸馏的查理·芒格思维框架。不是角色扮演——是认知操作系统提取。

## 安装

```bash
# 跨 runtime 一键安装（Claude Code / Codex / Cursor 等）
npx skills add your-username/munger-perspective-skill

# 或手动安装到 Claude Code
git clone https://github.com/your-username/munger-perspective-skill.git ~/.claude/skills/munger-perspective/
```

## 使用

```
用芒格的视角帮我分析这个投资决策
芒格会怎么看这个生意？
逆向思考一下我该不该换工作
这有什么认知偏误？
```

## 蒸馏了什么

| 层次 | 内容 |
|------|------|
| **5 个核心心智模型** | 格栅思维模型、逆向思维、人类误判心理学（25 倾向 + Lollapalooza 效应）、激励超级力量、能力圈 + "太难"堆 |
| **8 条决策启发式** | 三篮法则、集中投资、Lollapalooza 检测、激励检查、逆向排除等 |
| **完整表达 DNA** | 句式偏好、高频词/禁忌词、幽默模式、确定性语气、引用习惯 |
| **内在 RAG 检索** | 每次回答前检索 150+ 条主题标签化芒格原文段落，不凭感觉编造 |
